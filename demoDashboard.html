Magic Deploy


Proyecto para realizar el proceso de delivery continuo de proyectos WEB. Principalmente orientado a la publicacion de APIs desarrolladas en .NET en IIS, el proyecto consta de una aplicacion de escritorio a la cual se le configura un repositorio GIT y un determinado branch a monitorear. El sistema detecta nuevos commits en el branch y dispara el proceso de publicacion (clone-build-publish-backup de version actual-stop IIS-reemplazo con la nueva version-start IIS-eliminar archivos temporales).
